Expando

El objecto Expando es un componente que muestra más información a partir de un titular (h4). Puede recibir como argumento un objeto de propiedades para modificar su comportamiento por default y para los callbacks.

Ejemplo del objeto de configuración:

		{
			open: true,
			callbacks: {
				show: function() {},
				hide: function() {}
			}
		}

Realizando una selección con jQuery podemos asignarle una configuración de la siguiente forma:

        $("#Foobar").expando();

Cada vez que se instancia un componente Expando, retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

		var foo = $("#Foobar").expando();
        
		// returns
		foo = {
			uid: 121,                         //  Global number instance
			element: HTMLInputElement,        //  Input DOM Interface
			type: "expando ",                 //  Component Type
			open: boolean,				  	  //  Default behavior
			show(),						  	  //  Show dropdown
			hide()						  	  //  Hide dropdown
		}
