Modal Window

El modal es disparado por un link o botón del tipo submit (dentro de un form). Esto puede ser en el evento click, o bien, utilizando su método público.

Carga su contenido tanto por ajax como por parametro. Por defecto, lo hace por ajax tomando la url del elemento HTML que lo incia (en caso de que sea un link, levanta el href y si es un botón de tipo submit, levatan el action del form) o se le puede indicar una URL.

Por otra parte, el modal ofrece una clase de CSS closeModal la cual se la puede agregar a cualquier elemento HTML para que lo cierre.

Ejemplo del objeto de configuración:

		{
			content: string,
			width: number,
			height: number,
			ajax: boolean | string,
			onShow: function() {},			  // Callback executed on modal show
			onHide: function() {},			  // Callback executed on modal hide
			onContentLoad: function() {},	  // Callback executed on content load success (ajax includes)
			onContentError: function() {}	  // Callback executed on content load fail (ajax only)
		}

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

        $("#Foobar").modal();

Pasandole un objecto de configuración:

        $("#Foobar").modal({
        	content: "
Lorem...
",
        	width: 350,
        	height: 300
        });

Otra forma de iniciarlo es utilizando la siguiente sintaxis:

        $("#Foobar").modal("
Lorem...
");
        o
        $("#Foobar").modal("http://www.mercadolibre.com");

Cada vez que se instancia un componente Modal Window retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

		var foo = $("#Foobar").modal();
        
		// returns
		foo = {
			uid: 121,                         //  Global number instance
			element: HTMLInputElement,        //  Input DOM Interface
			type: "modal",                    //  Component Type
			content: string,				  //  Get content
			position(),					  	  //  Get, set and refresh position
			show(),						  	  //  Show modal
			hide()						  	  //  Hide modal
		}

El método público position() puede ser utilizado para:

1. Obtener la configuración actual de posicionamiento.

		foo.position();						  //  Get position

2. Actualizar la posición del elemento.

		foo.position("refresh");			  //  Refresh position

3. Setear una nueva configuración para el posicionamiento del elemento.

		foo.position({ 						  //  Set new position			
			context: $(element),
			offset: string,
			points: string,
			fixed: boolean
		});
