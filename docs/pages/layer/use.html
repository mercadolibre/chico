Contextual Layer

El objecto Contextual Layer pertenece a la familia Floats (al igual que Modal window y Tooltip). Recibe como argumento un objeto de propiedades para su funcionamiento, contenido o para los callbacks.

Ejemplo del objeto de configuración:

		{
			content: string,
			ajax: string,
			event: string,
			onShow: function() {},    		  // Callback executed on layer show 
			onHide: function() {},			  // Callback executed on layer hide
			onContentLoad: function() {},	  // Callback executed on content load success (ajax includes)
			onContentError: function() {} 	  // Callback executed on content load fail (ajax only)
			offset: string,					  // Positioner configuration. Default value "0 10".
			points: string,					  // Positioner configuration. Default value "lt lb".
			hold: boolean					  // Positioner configuration. Default value "false".
		}

Realizando una selección con jQuery podemos asignarle una configuración de la siguiente forma:

        $("#Foobar").layer({
        	content: "Lorem ipsum dolor sit amet, consectetur adipiscing elite.",
        	event: "click"
        });

Cada vez que se instancia un componente Contextual Layer retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

		var foo = $("#Foobar").layer();
        
		// returns
		foo = {
			uid: 121,                         //  Global number instance
			element: HTMLInputElement,        //  Input DOM Interface
			type: "layer",                    //  Component Type
			content: string,				  //  Get content
			position(),					  	  //  Get, set and refresh position
			show(),						  	  //  Show layer
			hide()						  	  //  Hide layer
		}

El método público position() puede ser utilizado para:

1. Obtener la configuración actual de posicionamiento.

		foo.position();						  //  Get position

2. Actualizar la posición del elemento.

		foo.position("refresh");			  //  Refresh position

3. Setear una nueva configuración para el posicionamiento del elemento.

		foo.position({ 						  //  Set new position			
			context: $(element),
			offset: string,
			points: string,
			fixed: boolean
		});
