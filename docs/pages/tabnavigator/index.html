Para iniciar este ejemplo se utilizó:

$('.demoTabNavigator').tabNavigator();

¿Cómo se usa?

El Tab Navigator se crea a partir de agregarle funcionalidad a un conjunto de anclas.

Cada vez que se instancia un Tab Navigator, éste retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

        var foo = $('.demoTabNavigator').tabNavigator(); // Same example
        
        // returns
        foo = {
            uid: 121,                         //  Global number instance
            element: HTMLInputElement,        //  Input DOM Interface
            type: "tabNavigator",             //  Component Type
            tabs: array,                      //  Instances of tabs
            select()                      	  //  Select a tab
        }

El Javascript para iniciarlo es:

$(".ajaxTabNavigator").tabNavigator();
