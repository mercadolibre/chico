¿Cómo se usa?

El tooltip es un componente emergente que puede iniciarse con cualquier elemento HTML que tenga el atributo title. Se dispara con el evento mouseover del elemento, o bien, utilizando su método público.

Ejemplo del objeto de configuración:

		{
			callbacks: {
				show: function() {},
				hide: function() {}
			}
		}

Realizando una selección con jQuery podemos asignarle una configuración de la siguiente forma:

        $("#Foobar").tooltip();

Cada vez que se instancia un componente Tooltip retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

		var foo = $("#Foobar").tooltip();
        
		// returns
		foo = {
			uid: 121,                         //  Global number instance
			element: HTMLInputElement,        //  Input DOM Interface
			type: "tooltip",                  //  Component Type
			content: string,				  //  Get content
			position(),					  	  //  Get, set and refresh position
			show(),						  	  //  Show tooltip
			hide()						  	  //  Hide toolip
		}

El método público position() puede ser utilizado para:

1. Obtener la configuración actual de posicionamiento.

		foo.position();						  //  Get position

2. Actualizar la posición del elemento.

		foo.position("refresh");			  //  Refresh position

3. Setear una nueva configuración para el posicionamiento del elemento.

		foo.position({ 						  //  Set new position			
			context: $(element),
			offset: string,
			points: string,
			fixed: boolean
		});
