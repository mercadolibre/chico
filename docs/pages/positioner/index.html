		});

			context: $("#context2"),
			points: "cm cm"				//  Element center-middle axis equal to Context center-middle axis
		});

¿Cómo se usa?

El Positioner es un componente que tiene como función posicionar elementos con respecto a un contexto.

El contexto puede ser cualquier elemento del DOM o el area visible de la ventana del navegador (viewport).

Existen 2 maneras de utilizar el componente, como objeto global o desde una instancia de componente:

		ui.positioner();				//  Simple sintaxis
		component.position();			//  Through component

Ejemplo del objeto de configuración:

		{
			element: $(element),		//  Element to be positioned
			context: $(element),		//  Context from which the element is positioned. Default: viewport.
			offset: string,				//  A string with X and Y coordinates (in pixels) separated by space (i.e.: "10 0").
			points: string,				//  A string with element and context shared axis, separated by space (i.e.: "lt lb").
			hold: boolean				//  Hold element's position.
		}

Realizando una selección con jQuery podemos asignarle una configuración de la siguiente forma:

        $("#Foobar").position({
        	context: $(".context"),
        	offset: "10 0",
        	points: "cm cm"
        });

ó desde el objecto público:

        ui.positioner({
        	element: $("#Foobar"),
        	context: $(".context"),
        	offset: "10 0",
        	points: "cm cm"
        });
