Para iniciar este ejemplo se utilizó:

$('.demoViewer').viewer();

¿Cómo se usa?

El Viewer se genera a partir de extender la funcionalidad de un listado de imágenes, las cuales poseen un link a una imagen de mayor tamaño.

Ejemplo del objeto de configuración:

		{
			callbacks: {
				select: function() {}
			}
		}

Cada vez que se instancia un Viewer, éste retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

        var foo = $('.demoViewer').viewer(); // Same example
        
        // returns
        foo = {
            uid: 121,                         //  Global number instance
            element: HTMLInputElement,        //  Input DOM Interface
            type: "viewer",                   //  Component Type
            children: array,                  //  Instances of components controlled by Viewer
            select()                      	  //  Select an item of main thumbnails
        }
