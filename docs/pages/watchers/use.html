Watchers (a.k.a. Validators)

El componente Watcher es la clase que define a todos los componentes de validación. Se asocia a uno o más controles de formularios y se encarga agrupar las validaciones por cada uno.

Cada instancia de Watcher posee una referencia directa con el elemento <form> padre. Cada elemento de formulario debería estar relacionado con una instancia del componente Forms de la familia Controllers. De esta manera logramos una referencia circular desde un elemento de formulario, la instancia de Watcher, el elemento <form> y su instancia de Forms.

Existen varias interfaces que permiten la utilización de los Watchers. Se dividen en 3 grupos: String, Number y Required. Cada uno de ellos define internamente un Watcher con configuraciones especiales de validación.

Lista de interfaces disponibles:

  • $.required()
  • $.string()
  • $.email()
  • $.url()
  • $.maxLength()
  • $.minLength()
  • $.Number()
  • $.max()
  • $.min()
  • $.price()
