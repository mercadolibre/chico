How to install

¿Cómo se usa?

Step 1: Minimized sources

You can find the source files in /build/. Or, if you prefer, you can grab them from here:

  • Production 29kb
  • Development 41kb
  •  

Content of /build/ folder:

		 /build/
		      chico-0.x.x.js          // dev
		      chico-0.x.x.css
		      chico-min-0.x.x.js      // prod
		      chico-min-0.x.x.css

Step 2: Browser sniffing

We hate writting CSS hacks, they are hard to maintain and kind of ugly. Instead of hacking the code, we rather use IE's conditional comments so if you paste this code in your HTML, Chico-UI will know how to fix the visual inconsistencies:

<!doctype html>





 <html lang="es" class="no-js"> 
<head>
