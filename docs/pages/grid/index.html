¿Cómo se usa?

El sistema de grillas de ChicoUI permite marcar un layout fluído muy rápido y con muy poco esfuerzo.

Dependencias

grid.css no está incluído en el core, por lo que debe ser llamada explícitamente.

Nomenclatura

Cada columna está expresada en fracciones que representan un porcentaje del ancho de nuestro contenedor.
La nomenclatura reemplaza la barra ( / ) en la fracción por un guión ( - ).

  • Ejemplo:
  • 50% = 1/2 = ch-g1-2
  • 75% = 3/4 = ch-g3-4
  • 20% = 1/5 = ch-g1-5

Calles

Las grillas no tienen paddings ni margins. Para poder generar las calles, tenemos que asignarle un márgen al elemento que está dentro de la grilla.
Para eso definimos 3 clases que le asignan márgenes de 10px al elemento correspondiente:

  • .leftcolumn: se le debe asignar al elemento de la izquierda
  • .centercolumn: se le debe asignar al elemento central
  • .rightcolumn: se le debe asignar al elemento de la derecha

Redondeo

Las columnas de valor 1/3 y 1/6 tienen anchos con decimales infinitos o demasiados largos.
Para solucionar este problema, creamos una grilla con el porcentaje extra que se necesita para redondear a 100%.

Si deseamos 3 columnas iguales, una de las 3 grillas debe ser ch-g1-3e de la siguiente manera:


	

		33,33%
	





	

		33,34%
	





	

		33,33%

Lo mismo ocurre con 1/6: una de las 6 grillas debe ser ch-g1-6e.

Si por alguna razón se necesita limpiar los floats, podemos usar la clase clearfix definida en core.css.
