About

It's the MercadoLibre's open source visual framework. The main goal is to improve and accelerate the development process. Builded on top of jQuery, this toolbox brings a couple of known visual patterns for the web and takes advantages of the new HTML5 and CSS3 features.

Chico-UI team is available for any feedback or suggestions. We want to share this tool with all designers and developers to make the web beautiful.

Give it a try!
