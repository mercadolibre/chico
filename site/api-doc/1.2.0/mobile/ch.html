ch (Namespace)

Description

An object which contains all the public members. A short alias for el.querySelectorAll

How-to

{% highlight javascript %} // Get all first level headings var headings = ch('h1'); // Get a list of p children elements under a container, whose parent is a div that has the class 'wrapper' var paragraphs = ch('p', ch('div.wrapper')); // The same as above var paragraphs = ch('p', 'div.wrapper'); {% endhighlight %}
