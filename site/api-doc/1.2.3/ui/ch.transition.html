ch.Transition

Description

Transition lets you give feedback to the users when their have to wait for an action.

How-to

{% highlight javascript %} // Create a new Transition. var transition = new ch.Transition($el, [options]); {% endhighlight %}
{% highlight javascript %} // Create a new Transition with jQuery or Zepto. var transition = $(selector).transition([options]); {% endhighlight %}
{% highlight javascript %} // Create a new Transition with disabled effects. var transition = $(selector).transition({ 'fx': 'none' }); {% endhighlight %}
{% highlight javascript %} // Create a new Transition using the shorthand way (content as parameter). var transition = $(selector).transition('http://ui.ml.com:3040/ajax'); {% endhighlight %}

Parameters

  • $el - jQuerySelector | ZeptoSelector : A jQuery or Zepto Selector to create an instance of ch.Transition.
  • options - Object : Options to customize an instance.
    • addClass - String : CSS class names that will be added to the container on the component initialization.
    • fx - String : Enable or disable UI effects. You must use: "slideDown", "fadeIn" or "none". Default: "fadeIn".
    • width - String : Set a width for the container. Default: "50%".
    • height - String : Set a height for the container. Default: "auto".
    • shownby - String : Determines how to interact with the trigger to show the container. You must use: "pointertap", "pointerenter" or "none". Default: "pointertap".
    • hiddenby - String : Determines how to hide the component. You must use: "button", "pointers", "pointerleave", "all" or "none". Default: "none".
    • reference - jQuerySelector | ZeptoSelector : It's a reference to position and size of element that will be considered to carry out the position. Default: ch.viewport.
    • side - String : The side option where the target element will be positioned. Its value can be: "left", "right", "top", "bottom" or "center". Default: "center".
    • align - String : The align options where the target element will be positioned. Its value can be: "left", "right", "top", "bottom" or "center". Default: "center".
    • offsetX - Number : Distance to displace the target horizontally. Default: 0.
    • offsetY - Number : Distance to displace the target vertically. Default: 0.
    • position - String : The type of positioning used. Its value must be "absolute" or "fixed". Default: "fixed".
    • method - String : The type of request ("POST" or "GET") to load content by ajax. Default: "GET".
    • params - String : Params like query string to be sent to the server.
    • cache - Boolean : Force to cache the request by the browser. Default: true.
    • async - Boolean : Force to sent request asynchronously. Default: true.
    • waiting - String | jQuerySelector | ZeptoSelector : Temporary content to use while the ajax request is loading. Default: '<div class="ch-loading-large ch-loading-centered"></div>'.
    • content - jQuerySelector | ZeptoSelector | HTMLElement | String : The content to be shown into the Transition container. Default: "Please wait..."
