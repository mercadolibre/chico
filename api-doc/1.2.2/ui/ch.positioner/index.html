ch.Positioner

Description

The Positioner lets you position elements on the screen and changes its positions.

How-to

Parameters

  • options - Object : Configuration object.
    • target - jQuerySelector | ZeptoSelector : Reference to the element to be positioned.
    • reference - jQuerySelector | ZeptoSelector : It's a reference to position and size of element that will be considered to carry out the position. If it isn't defined through configuration, it will be the ch.viewport.
    • side - String : The side option where the target element will be positioned. You must use: "left", "right", "top", "bottom" or "center". Default: "center".
    • align - String : The align options where the target element will be positioned. You must use: "left", "right", "top", "bottom" or "center". Default: "center".
    • offsetX - Number : Distance to displace the target horizontally. Default: 0.
    • offsetY - Number : Distance to displace the target vertically. Default: 0.
    • position - String : Thethe type of positioning used. You must use: "absolute" or "fixed". Default: "fixed".
