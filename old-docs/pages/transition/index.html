¿Cómo se usa?

El objecto Transition es una interface del objecto Modal Window que puede ser utilizado para para dirigir al usuario a lugares externos al flujo, como otros sitios o también para gateways externos dentro de un mismo flujo.

Ejemplo del objeto de configuración:

		{
			content: string,
			width: number,
			height: number,
			onShow: function() {},			  // Callback executed on transition show
			onHide: function() {}			  // Callback executed on transition hide
		}

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

        $("#Foobar").transition();

Pasandole un objecto de configuración:

        $("#Foobar").transition({
        	content: "
Lorem...
",
        	width: 350,
        	height: 300
        });

Otra forma de iniciarlo es utilizando la siguiente sintaxis:

        $("#Foobar").transition("Lorem...");

Cada vez que se instancia un componente Transition Window retorna un objecto público de control que nos brinda información sobre la instancia y nos permite ejecutar algunos métodos públicos.

		var foo = $("#Foobar").transition();
        
		// returns
		foo = {
			uid: 121,                         //  Global number instance
			element: HTMLInputElement,        //  Input DOM Interface
			type: "transition",               //  Component Type
			content: string,				  //  Get content
			position(),					  	  //  Get, set and refresh position
			show(),						  	  //  Show transition
			hide()						  	  //  Hide transition
		}

El método público position() puede ser utilizado para:

1. Obtener la configuración actual de posicionamiento.

		foo.position();						  //  Get position

2. Actualizar la posición del elemento.

		foo.position("refresh");			  //  Refresh position

3. Setear una nueva configuración para el posicionamiento del elemento.

		foo.position({ 						  //  Set new position			
			context: $(element),
			offset: string,
			points: string,
			fixed: boolean
		});
